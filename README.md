# Best Sport Byrds

Професионална уеб платформа за управление на **спортни птици** — състезателни гълъби, други спортни птици, бойни петли и др.

## Функционалности

| Модул | Описание |
|--------|----------|
| **Регистрация** | Собственици, състезатели, развъдчици; специализация по вид птици |
| **Птици** | Пръстен, вид, пол, линия, статус, родители |
| **Птицарници** | Множество локации, GPS координати |
| **Родословно дърво** | До 5 поколения, оценка на инбридинг |
| **Развъждане** | Двойки, сезони, яйца и излюпване |
| **Здраве** | Ваксини, лечения, прегледи, напомняния |
| **Тренировки** | Дистанция, метео, участващи птици |
| **Състезания** | Полети, изложби, борби; резултати и скорост |
| **Абонаменти** | Безплатен / Стандарт / Pro; одобрение от админ |
| **Админ панел** | Потребители, планове, плащания, настройки |
| **Супер админ** | Администратори, системни настройки |

## Изисквания

- PHP 8.2+
- MySQL 8.0+
- Apache с `mod_rewrite` (или Nginx + PHP-FPM)
- За локални тестове: [Docker Desktop](https://www.docker.com/)

## Локално тестване (Docker)

```bash
cd sport_birds
docker compose up -d --build
```

1. Отворете **http://localhost:8080/install**
2. База данни: хост `db`, потребител `sport_birds`, парола `secret`, база `sport_birds`
3. Създайте супер админ (имейл и парола)
4. Влезте на **http://localhost:8080/login**

Спиране: `docker compose down`

## Инсталация на споделен хостинг

1. Качете целия проект (или само съдържанието извън `docker/` ако няма нужда).
2. Насочете **document root** към папката `public/`.
3. Създайте MySQL база и потребител в cPanel.
4. Копирайте `.env.example` → `.env` и попълнете DB данните и `APP_URL`.
5. Отворете `https://вашият-домейн/install` и завършете инсталацията.
6. След инсталация: **забранете достъп до `/install`** (изтрийте route или защитете с парола).

### .htaccess на shared hosting

Ако приложението е в подпапка, редактирайте `public/.htaccess`:

```apache
RewriteBase /подпапка/
```

## Роли

- **user** — стандартен достъп според плана
- **admin** — управление на потребители, абонаменти, планове
- **super_admin** — пълен достъп + създаване на админи

## Планове (по подразбиране)

- **Безплатен** — 10 птици, 1 птицарник
- **Стандарт** — развъждане, тренировки, състезания
- **Професионален** — неограничено, родословна, експорт

Плащанията се заявяват от потребителя и се **одобряват ръчно** от админ (банков превод / Revolut).

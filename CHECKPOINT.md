## Локален тест

```bash
docker compose up -d
# http://localhost:8080/install
# DB: host db, user sport_birds, pass secret, database sport_birds
# Среда: Локална + Debug
```

След инсталация:
1. Регистрация / вход като супер админ
2. Птицарник → Птица (със снимка) → Родословна → Печат
3. Здраве, тренировки, състезания (активирай Pro от админ за пълен достъп)
4. Абонамент — заявка → одобрение в `/admin/subscriptions`

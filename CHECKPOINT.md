## Локален тест

```bash
docker compose up -d
# Миграцията към 2.0.0 се пуска автоматично при зареждане (ако има .env)
powershell -File scripts/local-install.ps1
```

**GPS тест (след вход):**
1. Регистрирай GPS устройство → копирай API токен
2. `curl -X POST http://localhost:8080/api/gps/track -H "Content-Type: application/json" -d "{\"token\":\"ТОКЕН\",\"latitude\":42.15,\"longitude\":24.75}"`
3. Отвори **Карта** или детайли на устройството
